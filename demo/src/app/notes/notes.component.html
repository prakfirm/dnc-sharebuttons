Notes

  • All social networks expect your site to have the Open Graph meta tags

    If you are using a meta tag module such as ngx-meta, ng2-meta, angular meta service ...etc, it will not help the social network to recognize the meta tags, because they update them using javascript, and these social networks do NOT support javascript execution yet.

    The inputs [title] [description] [image] behave differently for each social network, for example in facebook and pinterest they override the meta tags but some other social networks only accept the meta tags like google+ so the ideal solution is to make your app SEO friendly (using a server side rendering, like Universal)
