Customization & Style

Styling share buttons: disable the default style with [defaultStyle]="false" then use the classes shown in the style tab below to override them with your custom css.

Change buttons order: use the css rule order

Custom button template: use the input of the social network you want, e.g. give facebook a custom template [facebook]="facebookTemp".

Disable a button: pass 

false
to the social network input. e.g. [reddit]="false"
